## 06:54 设计新晋领班沟通与团队关系建设课程

**用户需求**：针对储备领班的两个痛点设计培训课程
1. 刚做领班，还没和团队建立正向合作关系，不敢提要求，怕矛盾
2. 缺乏更好的沟通方法和技巧

**设计思路**：
- 结合知识库现有《如何激励团队》PDF（情感账户理论 + 5把火框架）
- 结合《前场领班角色认知》PPT（6大职责、4大指标）
- 融入聚满分门店真实场景（前场/后厨/裱花、班前会、高峰期、稽核等）
- 强调实操话术，不是泛泛而谈的管理学

**课程名称**：《从同事到领班：建立正向团队关系与高效沟通》
**时长**：3小时（理论2h + 实战演练1h）
**五大模块**：
1. 破冰 + 身份转换（情感账户理论，3种管理者类型对号入座）
2. 点燃团队士气的"5把火"（每把火配门店场景+具体动作+话术）
3. 4个高频沟通场景实战（班前会派任务、现场纠偏、处理抱怨、跨部门协调）
4. 角色扮演演练（4组场景，评分维度）
5. 收尾 + 30天行动清单

**核心设计亮点**：
- "先存钱，后取款"——情感账户有余额，提要求才有底气
- 三明治法则（肯定→纠正→支持）——门店版4个高频场景话术模板
- 协调不是"命令"，是"共同解决问题"——跨部门沟通公式
- 30天行动清单：第1-7天每天关怀1人，第8-14天每天三明治纠偏1次...

**配套物料**：
- 讲师PPT大纲（17页结构）
- 学员手册（1页A4，话术速查卡+行动清单+情感账户记账表）
- 话术卡片4张（塑封可重复使用）

**与现有考核体系对接**：
- 情感账户/5把火 → 值班管理考核（班前会组织、团队氛围）
- 建设性反馈 → 训练四步骤考核（辅导技巧）
- 跨部门协调 → 管理制度考核（前场互动）
- 员工抱怨处理 → 客诉处理考核（沟通技巧）

## 07:16 课程方案升级版（理论深化 + 工具系统化）

**用户反馈**："不一定要用现有资料，有更好的就用更好的"

**升级思路**：
- 保留原版"情感账户"和"5把火"的实用性
- 引入更系统的管理沟通理论，解释"为什么不敢"的深层心理机制
- 从"话术模板"升级到"完整沟通脚本"，每套脚本都有理论来源

**核心理论升级**：
1. **Radical Candor**（Kim Scott）——直接挑战 + 个体关怀，解决"不敢提要求"
2. **Psychological Safety**（Google Aristotle项目）——高绩效团队第一特征，让团队敢说真话
3. **非暴力沟通NVC**（Marshall Rosenberg）——观察→感受→需要→请求，把命令变请求
4. **GROW模型**（John Whitmore）——辅导不是给答案，是帮员工自己找方法
5. **DESC脚本**（Bower & Bower）——困难对话的标准化表达
6. **SBI反馈公式**（CCL）——情境→行为→影响，让反馈可复制
7. **权力悖论**（Dacher Keltner）——解释晋升后反而更难开口
8. **LMX理论**——领导-成员交换关系分层，理解"陌生人阶段"

**课程结构升级**：
| 维度 | 原版 | 升级版 |
|:---|:---|:---|
| 时长 | 3h | 4h（理论2.5h + 演练1.5h）|
| 理论深度 | 情感账户 + 5把火 | Radical Candor + 心理安全感 + LMX + 权力悖论 |
| 工具 | 零散话术模板 | 4套完整脚本（NVC / GROW / DESC / SBI），每套有理论来源 |
| 课前 | 无 | 课前诊断问卷（关系质量自评 + 心理安全感自测）|
| 课后 | 30天行动清单 | 30天行动清单 + 情感账户记账表 + 60天回访设计 |
| 演练 | 4组场景 | "你最头疼的一刻"真实场景复活赛 |
| 配套 | PPT大纲 + 手册 + 卡片 | PPT大纲 + 4页手册 + 塑封卡片 + 师傅带训指南 |

**两个版本并存**：
