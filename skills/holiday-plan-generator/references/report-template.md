## 报告生成模板（钉钉友好格式，最新版）

生成报告时遵循以下结构和格式规则：

### HTML截图字体规格（手机直接看，无需放大）
- 正文：22px
- 模块标题：32px
- 班次标题（早班/晚班/后厨/裱花房）：24px
- 目标金额：56px
- 行高：2.1
- 模块间距加大

### 格式规则
- 禁止 markdown 表格（| 列1 | 列2 |）
- 禁止减号列表（- 项目）
- 使用 **加粗** 括关键信息
- 每段之间空一行
- 使用数字列表
- 用 emoji 分隔模块
