# 《值班管理》带训实操指引

## 课程定位

| 项目 | 内容 |
|------|------|
| **归属岗位** | 储备领班 / 储备店长 |
| **学时安排** | 理论1h + 实操贯穿整个带训周期（建议第10-25天重点突破） |
| **学成标准** | 能独立盯一个完整班次，处理常见问题，班次结束后能复盘 |
| **关联课程** | 训练四步骤（前置）、当班管理、团队激励 |

---

## 一、师傅带训流程（分5步）

### Step 1：自己先学（带训前，师傅做）

**目标**：师傅自己能讲清楚值班管理是什么、为什么、怎么做。

| 动作 | 具体内容 | 验收标准 |
|------|---------|---------|
| 学课程 | 在满分学院学完《值班管理》全部视频和文档 | 能不看资料复述课程框架 |
| 看评估表 | 熟悉值班管理模块的评估表打分项 | 知道哪些是"关键项"（不过就挂） |
| 复盘自己 | 回想自己当年学值班管理时踩过什么坑 | 能讲出2-3个真实案例 |
| 备徒弟 | 了解徒弟目前水平——有没有盯过班？处理过突发情况？ | 心里判断徒弟是"零基础"还是"有点经验" |

---

### Step 2：讲给他听（第1次接触，约30分钟）

**目标**：让徒弟知道"值班管理"到底管什么，不是简单的"站在那看着"。

**师傅这样做**：

1. **画一张值班管理的"地图"给徒弟看**

   ```
   值班管理 = 班前 → 班中 → 班后
   
   班前：人员到岗检查 → 物料准备确认 → 设备检查 → 目标宣导
   班中：现场巡视 → 出品监控 → 人员调度 → 客诉处理 → 突发事件
   班后：数据记录 → 交接事项 → 复盘总结
   ```

2. **讲一个你自己的翻车案例**
   
   > "上次我值班时没检查冰箱温度，结果下午高峰期发现某原料变质，临时换菜单，损失XX元。值班管理不是走形式，每个检查点都是血的教训。"

3. **明确告诉徒弟这课的考核标准**
   
   > "学完这课，我要你能独立盯一个班，从开班到打烊，我全程不插手。考核时我会用评估表，这几个是生死线：
   > - 班前检查是否漏项
   > - 高峰期有没有主动调度人员
   > - 班后有没有做复盘记录"

---

### Step 3：做给他看（第2-3次，师傅示范）

**目标**：徒弟亲眼看到"标准的值班管理"长什么样。

**师傅带一次完整班次，让徒弟跟着看，每步讲解：**

| 时段 | 师傅做什么 | 同时讲解什么 |
|------|-----------|-------------|
| **班前20分钟** | 按检查表逐项确认：人员到岗、物料备货、设备状态、清洁状况 | "为什么要先查设备——因为高峰期设备坏了，你根本没空修" |
| **开班前5分钟** | 开简短班前会：今日目标、人员分工、注意事项 | "班前会不超过3分钟，讲重点，大家站着听，不说废话" |
| **班中（高峰期）** | 不停走动巡视，看出品速度、看产品质量、看人员状态 | "高峰期你的眼睛要在全场扫，不是盯着一个地方" |
| **突发情况** | 遇到客诉/缺货/人员不适，当场处理 | "先解决客人的问题，再追责。客人走了就没机会了" |
| **班后15分钟** | 填写值班记录，和下一班交接，做简单复盘 | "复盘不用写小作文，三个问题：今天什么问题？怎么解决的？明天怎么避免？" |

**关键动作**：
- 每做完一步，停下来问徒弟：**"我刚才为什么先做X，再做Y？"**
- 让徒弟拿手机拍下班前检查表、值班记录表的样子，回去自己对着看。

---

### Step 4：让他做，你看（第4-6次，徒弟实操）

**目标**：徒弟上手，师傅在旁观察，不插手，除非重大失误。

**布置具体任务**：

| 轮次 | 徒弟任务 | 师傅观察重点 | 反馈时机 |
|------|---------|-------------|---------|
| 第1次盯班 | 负责班前检查和班前会 | 检查有没有漏项、班前会是否讲清重点 | 班后立即 |
| 第2次盯班 | 负责整个班次，师傅在旁看 | 高峰期会不会主动走动、会不会调度人员 | 班后立即 |
| 第3次盯班 | 独立盯班，师傅不站旁边，但随时可找 | 遇到突发情况怎么处理、班后有没有复盘 | 当天收班后 |

**反馈方法（班后立即，不超过15分钟）**：

用评估表逐项过，**先问徒弟自己觉得怎么样，再给师傅评价**：

```
师傅："今天这个班，你自己觉得哪里做得最好？"
徒弟："我觉得班前检查我做得比较细..."
师傅："对，班前检查确实没漏项。那哪里觉得没做好？"
徒弟："高峰期有个员工请假，我临时调配有点乱..."
师傅："是的，我当时看你在那想了十几秒才安排。下次可以更快——A岗位缺人，先看B岗位谁有空，直接调，不要犹豫。
      评估表上我给你打了X分，班前检查通过，人员调度这一项还差一些，下次重点练。"
```

---

### Step 5：让他独立做，你验收（第7次起，出师考核）

**目标**：徒弟独立盯班，师傅用评估表正式打分。

| 考核项目 | 通过标准 | 关键项（不过即挂） |
|---------|---------|------------------|
| 班前准备 | 检查表全部完成，无漏项 | ★ 设备/安全检查无遗漏 |
| 班中会管理 | 主动巡视≥3次/小时，能处理常见问题 | ★ 高峰期出品质量无客诉 |
| 人员调度 | 遇缺员能5分钟内重新分配 | ★ 无岗位真空（没人管的情况） |
| 班后复盘 | 填写值班记录，做口头或书面复盘 | ★ 有记录、有总结 |
| 突发应对 | 遇到客诉/缺货能按流程处理 | ★ 无客人升级投诉 |

**考核方式**：
1. 徒弟独立盯一个完整班次
2. 师傅不站旁边，但可以远距离观察（或事后看监控）
3. 班后30分钟内，师傅用评估表逐项打分
4. **关键项全部通过 + 总分≥80分 = 出师**
5. 关键项有挂的，针对该项补训2-3次，再考

## 二、徒弟学习路径（《值班管理》专项）

### 第1阶段：看懂（第1-2天）

| 动作 | 内容 | 交付物 |
|------|------|--------|
| 看课程 | 满分学院《值班管理》视频+文档 | 看完 |
| 做笔记 | 画出值班管理的流程图（班前-班中-班后） | 手写或手机备忘录 |
| 做理论考 | 满分学院线上理论考核 | 通过截图 |
| 问问题 | 把看不懂的点列出来，问师傅 | 问题清单（至少3个） |

### 第2阶段：看会（第3-5天）

| 动作 | 内容 | 交付物 |
|------|------|--------|
| 跟师傅盯班 | 跟着师傅看2个完整班次，每步记录师傅在做什么 | 观察笔记（师傅每步动作+目的） |
| 拿检查表 | 向师傅要一份班前检查表、值班记录表，拍照保存 | 手机相册里有标准模板 |
| 问为什么 | 每看一步，心里问"为什么先做这一步"，不懂就问 | 问题清单 |

### 第3阶段：上手（第6-15天）

| 动作 | 内容 | 交付物 |
|------|------|--------|
| 从班前开始 | 第1次盯班只做班前检查，师傅验收 | 班前检查表（自己打勾） |
| 逐步加任务 | 第2次盯班做班前+班中会巡视，第3次做全流程 | 每次师傅的评估表 |
| 每周总结 | 每周一写学习总结发钉钉群，重点写值班管理的学习 | 钉钉群消息 |
| 主动要反馈 | 每次盯班后主动问师傅"今天哪里做得不好" | 师傅的口头反馈记录 |

### 第4阶段：独立（第16-20天）

| 动作 | 内容 | 交付物 |
|------|------|--------|
| 独立盯班 | 师傅不站旁边，自己独立盯完整班次 | 值班记录表 |
| 处理突发 | 遇到问题时先自己处理，处理完告诉师傅 | 事件处理记录 |
| 班后复盘 | 每次下班做3分钟复盘：问题-解决-避免 | 复盘笔记 |
| 接受考核 | 师傅用评估表正式考核 | 评估表 |

## 四、常见问题 & 师傅怎么应对

| 徒弟问题 | 师傅应对 |
|---------|---------|
| "我跟着你看了两次，但自己一上手就慌" | 正常。前3次紧张是对的。第1次只让你做班前检查，不让你管全流程，逐步加任务。 |
| "高峰期事情太多，我不知道先看哪个" | 教你优先级：安全（人/设备）> 客诉 > 出品质量 > 效率。先保不出事，再保做得好。 |
| "员工不听我的，我是新来值班的" | 提前和班组打好招呼"今天XX学徒值班，大家配合一下"。徒弟也要学会用"店长让我今天值班"来建立权威，不是靠吼。 |
| "班后复盘不知道写什么" | 不用写小作文，就三个问题：今天出了什么问题？我怎么处理的？下次怎么避免？ |
| "理论考核过了，但实操还是不会" | 理论是知道，实操是做到。中间差的是"练"。安排多盯几次班，特别是高峰期。 |

---

## 五、一句话总结

> **师傅**：带《值班管理》，不是教"知识"，是带"习惯"——班前检查的习惯、走动巡视的习惯、班后复盘的习惯。习惯养成了，徒弟就会了。
>
> **徒弟**：学《值班管理》，不是背流程，是练反应——高峰期来了你慌不慌、出问题了你能不能处理。多盯几次班，比看十遍视频管用。
