# 培训试题生成器

根据用户提供的培训材料（PPT、Word、PDF或文本），自动生成符合标准格式的线上考试试题Excel文件。

## 工作流程

### 1. 理解需求
确认以下信息：
- **题目总数** 和 **题型分布**（如：20道题，单选10、多选5、判断5）
- **培训材料**：用户提供课件、文档或文字内容
- **试题分类**：试题所属分类名称（如"店长角色认知"）

如果用户未明确题型分布，默认建议：单选10道、多选5道、判断5道。

### 2. 读取培训材料
支持格式：
- **PPT** (.pptx) - 使用 `python-pptx` 读取
- **Word** (.docx) - 使用 `python-docx` 读取
- **PDF** (.pdf) - 使用 `pdfplumber` 读取
- **Excel** (.xlsx) - 使用 `openpyxl` 读取
- **文本** - 直接使用

读取后提取核心知识点，包括：
- 关键概念和定义
- 角色职责和要求
- 流程和标准
- 重要原则和理念

### 3. 生成试题

#### 出题原则（基于布鲁姆分类法）

**完整的试题结构**：

| 类型 | 比例 | 作用 | 示例 |
|------|------|------|------|
| **记忆类** | 33% | 帮助学员记住关键概念、方法论、建立管理框架 | 5把火包括哪些？三种管理风格是什么？情感账户的存款指什么？ |
| **理解类** | 17% | 理解概念内涵、逻辑关系、因果分析 | 为什么老好人型店长带不出铁军？严厉却服众型的成功关键是什么？ |
| **应用类** | 17% | 在具体情境中运用知识做决策 | 新员工连续失误，店长该怎么做？团队士气低落时优先做什么？ |
| **分析类** | 17% | 综合评估、对比辨析、案例判断 | 哪些做法符合建设性反馈？哪些行为会导致情感账户透支？ |
| **理念类** | 17% | 核心价值观认同、管理哲学内化 | 店长的成功取决于帮助员工成长。建立情感链接是高效管理的基础。 |

**出题前的知识梳理（避免遗漏）**：

生成试题前，先梳理培训材料中的**核心知识点清单**：
1. **关键概念**：如"情感账户"、"存款"、"取款"、"领导力本质"
2. **方法论框架**：如"5把火"的具体内容
3. **分类体系**：如"三种管理风格"的类型和特征
4. **核心原则**：如"先存款再取款"、"让下属成功"
5. **应用场景**：如"新员工失误"、"团队士气低落"、"员工培养"

**知识点覆盖检查表**：
- [ ] 每个核心概念都有对应的题目
- [ ] 方法论框架被完整考察（不遗漏任何要素）
- [ ] 分类体系的各个类型都被覆盖
- [ ] 重要原则在记忆/理解/理念类题目中体现
- [ ] 关键应用场景在应用类题目中体现

**避免知识点重复的规划方法**：

**同一知识点，不同角度考察**：

| 知识点 | 记忆类（是什么） | 理解类（为什么） | 应用类（怎么做） |
|--------|-----------------|-----------------|-----------------|
| 情感账户 | 存款/取款的定义 | 先存款再取款的逻辑 | 如何积累情感存款 |
| 5把火 | 5把火包括哪些 | 每把火的原理和作用 | 什么情境下使用哪把火 |
| 管理风格 | 三种风格的名称 | 严厉却服众vs遭抵抗的区别 | 如何判断自己属于哪种风格 |

**禁止的简单重复**：
- ❌ 同一知识点以相同角度出现两次
  - 例："情感账户的存款指什么？"（记忆） 和 "存款是指建立信任"（另一道记忆题）
- ❌ 只是换说法，实质相同的题目
  - 例："5把火包括哪些？" 和 "点燃团队士气的方法有哪些？"（内容完全重叠）

**允许的深度延伸**：
- ✅ 同一知识点，不同题型深化理解
  - 例：记忆题"5把火包括哪些？" + 应用题"团队士气低落优先使用哪把火？"
- ✅ 同一知识点，正反面考察
  - 例：记忆题"存款是什么？" + 分析题"哪些行为会导致透支？"

**题目去重检查清单**：
1. 列出所有题目的**考察知识点**和**考察角度**
2. 检查是否有**相同知识点+相同角度**的题目
3. 如有重复，合并或改为不同角度
4. 确保每个知识点最多出现**2次**（且角度不同）

**各类题目的设计要点**：

**🧠 记忆类题目（建立框架）**：
- 考察关键方法论：如"5把火包括哪些核心方法？"
- 考察核心概念：如"情感账户中的存款指的是什么？"
- 考察分类体系：如"三种店长管理风格类型是什么？"
- **目的**：帮助学员建立完整的知识框架，记住管理工具和方法

**💡 理解类题目（深化认知）**：
- 探讨因果关系：如"为什么严厉却服众型能成功？"
- 对比辨析差异：如"两种严厉型店长的核心区别是什么？"
- 解释核心理念：如"领导力的本质是让下属成功意味着什么？"
- **目的**：不仅记住，更要理解为什么

**🎯 应用类题目（指导实践）**：
- 真实工作情境：如"新员工连续失误，最有效的做法是？"
- 具体决策判断：如"团队士气低落，应该优先采取哪种方式？"
- **目的**：培训后能直接应用于实际工作

**🔍 分析类题目（综合判断）**：
- 多维度评估：如"哪些做法属于表达关怀？"
- 识别误区：如"哪些行为会导致情感账户透支？"
- **目的**：培养系统思维，能识别正确与错误做法

**💖 理念类题目（价值观塑造）**：
- 核心观念认同：如"店长的成功取决于帮助员工成长"
- 管理哲学内化：如"建立情感链接是高效管理的基础"
- **目的**：从思想上认同正确的管理理念

**题目质量自检清单**：
1. 这道题属于记忆/理解/应用/分析/理念中的哪一类？
2. 记忆类题目是否涉及**关键方法论和核心概念**（而非无关细节）？
3. 理解/应用/分析类题目是否能证明学员**真正理解了概念**？
4. 这道题在**真实工作场景**中是否有参考价值？
5. 错误选项是否具有**迷惑性**，能暴露常见误区？
6. **该知识点是否已在其他题目中以相同角度考察过**？

#### 题目设计要点
1. **题干清晰**：明确表达考察点，避免歧义
2. **选项设计**：
   - 单选题：1个正确选项，3个干扰项（共4个选项）
   - 多选题：**2-6个选项**，其中正确选项2-4个。当知识点涉及要素超过4个时，可扩展为5-6个选项（如：值班管理六大角色、班前会6步流程等），确保完整考察所有要素而不遗漏。
   - 判断题：陈述明确，答案无争议
3. **答案准确**：正确答案必须基于培训材料原文
4. **难度适中**：60%基础题、30%理解题、10%应用题

### 4. 输出Excel文件

使用模板格式（见 `assets/template.xlsx`），包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 试题所属分类 | 试题分类名称 | 店长角色认知 |
| 试题类型 | 单选题/多选题/判断题 | 单选题 |
| 题目 | 题干内容 | 店长与店员的关注重点有何不同？ |
| 选项A | 选项内容 | 店员关注个人任务... |
| 选项B | 选项内容 | 店长关注团队一起完成目标 |
| 选项C | 选项内容 | 店长和店员关注的内容完全相同 |
| 选项D | 选项内容 | 店长只关注营业额... |
| 选项E | 选项内容（多选题知识点超过4个时使用） | 店长关注顾客体验... |
| 选项F | 选项内容（多选题知识点超过5个时使用） | 店长关注设备保养... |
| 正确答案 | 正确选项字母 | B（单选）/ ABC（多选）/ 正确/错误（判断） |
| 参考答案 | 问答题答案 | - |
| 评分标准 | 实操题评分标准 | - |

#### Excel格式规则
- **单选题**：选项A-D必填，E、F填"-"，正确答案填单个字母（A/B/C/D）
- **多选题**：**选项A-F按需填写**（知识点超过4个时可扩展到E、F），正确答案填多个字母（如"ABC"、"ABCD"、"ABCDEF"）
- **判断题**：选项A-F全部填"-"，正确答案填"正确"或"错误"

### 5. 使用脚本生成

运行脚本生成试题Excel：

```bash
python3 scripts/generate_quiz.py --input <培训材料路径> --output <输出路径> --config <配置>
```

或直接调用函数生成（详见脚本内 `generate_quiz()` 函数）。

## 脚本使用

### generate_quiz.py

主要功能：
1. `read_training_material(path)` - 读取培训材料
2. `generate_questions(content, config)` - 根据内容生成试题
3. `create_excel(questions, output_path)` - 输出Excel文件

示例配置：
```python
config = {
    "category": "店长角色认知",
    "total": 20,
    "single_choice": 10,
    "multiple_choice": 5,
    "judgment": 5
}
```

## 质量检查

生成后确认：
- [ ] 题目数量正确
- [ ] 题型比例正确
- [ ] 答案与培训材料一致
- [ ] Excel格式符合模板要求
- [ ] 分类名称统一

## 输出示例

文件命名建议：`{培训主题}_试题.xlsx`

如：`店长角色认知_试题.xlsx`
