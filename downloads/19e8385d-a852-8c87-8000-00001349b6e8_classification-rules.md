# 报销单分类规则参考

## 一级类型（7个）
1. **原材料** — 门店生产所需的食材、原料
2. **差旅** — 外出产生的交通、住宿、餐饮等费用
3. **工资** — 员工薪酬相关
4. **租金水电物管** — 门店房租、水电、物业管理
5. **福利费** — 员工福利、团建、聚餐等
6. **办公** — 办公用品、文具、通讯、招聘等
7. **运输费** — 快递、物流、配送

## 分类判断优先级

1. **先匹配明确关键词** — 如"广东联合电子服务"直接判定为过桥费
2. **再匹配二级类型关键词** — 如"酒店"→住宿费
3. **最后确定一级类型** — 住宿费属于差旅
4. **无法判断时** — 标记为"待确认"，给出最可能的分类建议

## 常见模糊场景处理

| 场景 | 处理方式 |
|------|---------|
| 超市小票（多种商品混合） | 看主要商品类别，或拆分为多条（如果用户要求） |
| 外卖订单 | 看用途：员工餐→工作餐；加班餐→福利费/工作餐 |
| 加油票 | 明确为油费（差旅下） |
| 打车票 | 明确为车费（差旅下） |
| 网购截图 | 看商品类别：文具→办公文具；食材→原材料 |
| 无文字的图片 | 标记"无法识别"，请用户补充说明 |
| 只有金额没有用途 | 标记"待确认"，列出可能分类让用户选 |

## 金额提取规则

1. 找 ¥ / ￥ / 元 / RMB / CNY 符号附近的数字
2. 优先取最大金额（避免取数量×单价）
3. 如果有"合计""总计""应收"字样，取该金额
4. 交通费按里程计算时：1元/km，保留2位小数

## 日期提取规则

1. 找 YYYY-MM-DD / YYYY/MM/DD / MM-DD / MM/DD 格式
2. 没有日期的用截图中的时间戳，或标记"日期待补"
3. 统一输出格式：YYYY-MM-DD
