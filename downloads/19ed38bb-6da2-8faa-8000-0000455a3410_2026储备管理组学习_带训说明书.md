# 2026储备管理组学习&带训说明书

**适用对象：** 储备前场领班、储备后厨领班、储备店长及其带训师傅 / 门店店长 / 赋能优化部

---

## 一、培训目的和目标

本项目不是单纯"上课打卡，也不是为了考核而培训"，而是围绕门店真实经营场景，帮助储备管理组从会执行的员工，成长为能独立承担岗位责任的管理者。

### 目标维度与达成标准

| 目标维度 | 达成标准 |
|---------|---------|
| 岗位胜任 | 掌握岗位应会技能，能独立完成前场/后厨/店长关键管理动作 |
| 带人能力 | 掌握训练四步骤，能把标准讲清楚、示范清楚、纠偏清楚，并追踪到结果 |
| 现场管理 | 能围绕服务、品质、清洁、效率、成本、业绩目标开展班前、班中、班后管理 |
| 经营意识 | 储备店长需具备订货、损益、利润、团队激励、挑战日/周周PK等经营动作基本能力 |

---

## 二、培养人员提报-培训-考核-晋升流程

管理组培养采用"提报、培训、考核、晋升"的闭环流程，责任主体清晰，避免学员只学习不落地、门店只提报不带训。

**一句话目标：** 60天出结果 — 让储备管理组在60天内完成岗位学习地图、理论学习、岗位实操、师傅评估和赋能考核，达到"能独立上岗、能带人、能管现场、能用数据改善"的上岗标准。

### 流程节点

| 流程节点 | 责任方 | 关键动作 |
|---------|--------|---------|
| 1. 培养名单提报 | 营运店长 | 店长向赋能优化部陈晓腾提报培养名单 |
| 2. 培训启动会议 | 赋能部 | 组织储备管理组和师傅召开线上沟通会议，统一学习目标、带训方法、考核要求 |
| 3. 线下理论课程 | 赋能部 | 赋能优化部定期组织储备管理组到公司参加线下培训营 |
| 4. 岗位实操带训 | 店长/师傅 | 师傅按满分学院带训地图完成岗位带训，学员按要求完成考核与实操作业 |
| 5. 晋升考核申请 | 店长 | 学员完成任务并通过师傅/店长初考后，由店长在钉钉发起晋升考核申请流程。需提前至少一周申请，尽量避免月最后一周申请 |
| 6. 晋升考核实施 | 赋能部 | 接到申请后，由考核负责人组织到店实施理论、现场提问、实操考核（考核内容见《各岗位考核评估表》） |
| 7. 晋升申请 | 人力行政 | 考核通过后，门店按岗位要求提报晋升对象，人力资源部按流程审批，次月晋升生效 |

## 三、学什么？

每门课都对应真实工作场景，要求学完能用、用完能复盘。不同培养岗位的学习重点如下：

**重点课程：** 训练四步骤、订货管理、值班管理、后厨现场管理、损益/利润管理、赢得挑战日，是从员工转向管理者的关键能力。

| 培养岗位 | 核心学习内容 | 最终要能做到 |
|---------|------------|------------|
| 储备前场领班 | 企业文化、角色认知、前场事务、顾客满意度、陈列管理、食品安全、维护清洁、训练四步骤、订货&生产计划、区域值班管理、产品出品验收标准 | 会带人、会管货、会盯班，能保障服务、品质、卫生和业绩动作不断档 |
| 储备后厨领班 | 产品技能强化、企业文化、后厨领班角色认知、领班事务、食品安全、设备维护清洁、3S管理、产品验收标准、训练四步骤、后厨现场管理 | 强技能、会带人，能管控出品品质、效率、安全卫生和成本 |
| 储备店长 | 店长角色认知、行政事务、订货管理、优秀值班管理、团队激励、损益报表分析&应用、利润管理、赢得挑战日/周周PK | 会带人、会管货、会激励、会复盘，能带团队达成业绩和利润目标 |

## 四、学员如何学习？

学员学习遵循"看、考、写、练、过"五个动作。学习顺序不强制完全一致，但必须围绕岗位学习地图完成课程、考核和实操闭环。

| 动作 | 要求 | 交付物/证据 |
|-----|------|-----------|
| 看 | 按满分学院学习地图看视频、读文档，先理解标准和场景 | 平台学习进度 |
| 考 | 完成线上理论考核；不通过则重学、重考 | 理论考核通过记录 |
| 写 | 每周一次学习总结，发到管理组赋能培训群，不隐藏问题 | 周学习总结 |
| 练 | 配合师傅排班和任务安排，在真实班次中反复实操 | 实操作业、班次练习记录、评估表 |
| 过 | 完成师傅评估和赋能考核，争取一次通过 | 最终考核结果 |

**学习方法：** 知识学习 + 实操应用 + 及时反馈。只看课程不算学会，能在现场解决问题、被评估表验证通过，才算真正掌握。

## 五、师傅如何带训？

师傅不是"凭经验教"，而是按学习地图、评估表和真实岗位任务带训。带训质量直接影响学员60天能否出师。

### 带训储备管理组应有的态度

| 错误态度 | 正确态度 |
|---------|---------|
| 为考核而培训，走过场 | 以考核为标尺，提能力 |

### 储备管理组带训步骤

| 步骤 | 关键内容 |
|-----|---------|
| 定计划 | 与徒弟一起制定周/月培训计划。避免想到教什么就教什么的盲目 |
| 先吃透 | 师傅先看满分学院，确保"自己会" |
| 做示范 | 让徒弟看到标准长什么样 |
| 给实战 | 通过实际工作安排（挑战日、当班管理等）让徒弟实操 |
| 给反馈 | 观察→反馈→辅导→再练→再反馈，循环打磨 |
| 逐项查 | 实操过程中用考核表逐项检验，查漏补缺 |
| 验成果 | 每个项目都安排一次正式考核，用评估表评估，确保都能通过 |

### 非后厨出身师傅带训后厨领班步骤

| 步骤 | 关键内容 |
|-----|---------|
| 定计划 | 与徒弟一起制定周/月培训计划 |
| 先吃透 | 师傅先看满分学院，确保"自己知道标准，流程和方法"。不一定要会做 |
| 分配任务探讨方法 | 1、针对现在存在的问题，分配改善任务，明确目标。2、和徒弟一起，找出备料、分工、交接、设备、人员配合中的机会点 |
| 他做你看支持协助 | 师傅不做监工，做支持者。提供资源，解决徒弟做的过程中的困难和问题 |
| 复盘任务彻成效 | 和徒弟一起复盘，达标就沉淀标准，未达标就分析原因并重新调整计划，直到达成 |
| 逐项查 | 实操过程中用考核表逐项检验，查漏补缺 |
| 验成果 | 每个项目都安排一次正式考核，用评估表评估，确保都能通过 |

**说明：** 非后厨出身店长（师傅）不需要亲自教面包技术，而要扮演管理教练。重点带"标准、流程、目标、复盘"，帮助后厨领班从会做产品变成会管理后厨。

## 六、满分学院登录和操作指引

1. 打开【聚满分钉钉】，在工作台或应用入口点击【满分学院】
2. 进入登录页后，优先使用【手机号接受验证码】登录
3. 进入【培训考试】栏目，找到本人岗位学习地图
4. 点击学习地图右侧【去学习】，按课程要求完成视频、文档、理论考核和实操作业
5. 师傅也可以用同样的方式学习，及了解徒弟的学习内容

## 七、申请考核前需做好哪些准备？

为提高赋能考核一次通过率，建议门店在发起晋升考核申请前，先完成以下"提报前检查"。未完成的不建议直接提报，避免学员到考场才暴露基础薄弱。

| 检查项 | 提报前必须确认 | 通过率关键动作 |
|-------|-------------|-------------|
| 学习进度 | 满分学院岗位学习地图达到100%，阶段课程已完成 | 不要等最后一天集中刷课；每周跟进进度，低的及时拉齐 |
| 理论考核 | 对应课程理论考核均已通过 | 错题必须复盘，师傅抽问核心概念和关键标准 |
| 实操作业 | 平台实操作业、师傅阶段评估、关键评估表已提交 | 实操作业必须已通过师傅的评估 |
| 岗位实操 | 完成足够真实班次练习，并且店长过程中都有及时反馈，不断修正，直到能独立承担目标岗位关键场景 | 前场/店长重点看班前会、控场、订货、复盘；后厨重点看品质、效率、卫生、成本 |
| 周总结 | 学员每周有学习总结，两个月合计至少8次 | 总结要写真实问题和复盘，不写空话。能讲清学了什么、用在哪里、改善了什么 |
| 师傅初考 | 师傅/店长已完成初考，确认重点考核内容能熟练应用，评估分数能达85分以上 | 师傅/店长给徒弟做模拟考，考核要求要更严（用相应岗位评估表） |
| 门店验证 | 门店现场无明显红线问题，学员所在岗位标准执行稳定 | 食安、效期、品质、服务、清洁等底线问题先解决，再提报 |

**重点实操项目：**
- 储备前场领班：独立全程负责一个挑战日，并有书面的计划和复盘
- 储备店长：同上 + 一次损益表分析和行动计划
- 后厨领班：独立负责一个周末值班，能体现出品质监管，效率提升，成本管控等工作内容

**建议提报口径：** 学员已完成岗位学习地图、理论考核、实操训练和师傅初考；门店确认其能独立承担目标岗位核心工作，现申请赋能优化部组织晋升考核。

## 八、赋能到店考核流程

1. **时长：** 整个考核时长约为3小时左右
2. **开篇：** 介绍当天考核流程和内容
3. **问答：** 考核知识性的内容
4. **现场实操：** 训练四步骤，产品制作，班前会，交接班，巡视，订货，损益分析（储备店长）
5. **看监控考核：** 学员选挑战日当班最好的一天，和考核人员一起看监控考核【值班管理】相关内容
6. **总结：** 评分，辅导，公布结果

---

## 九、考核当天的需做什么准备？

1. **吃饭安排：** 如考核时间是饭点，学员提前安排吃饭
2. **人手安排：** 当天需把学员安排在营运人手之外（避免影响营运）
3. **资料准备：** 准备好挑战日实操时要求做的【行动计划】【复盘】文本（手写，电子的都可以）
4. **团队共识：** 通知门店所有伙伴，今天是该伙伴的考核日，一切工作积极配合

| 角色 | 必须做好的事 |
|-----|-------------|
| 学员 | 看课程、过理论、写周总结、做实操、接受反馈并改进 |
