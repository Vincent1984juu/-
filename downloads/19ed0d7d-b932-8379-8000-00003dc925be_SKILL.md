# 挑战日计划

## 工作流程

### 步骤1：触发识别
用户说"挑战日计划"、"挑战日"、"周六挑战日"、"周日挑战日"、"周末挑战日"等关键词时，触发本技能。

### 步骤2：发出指引（立即执行）
向用户发送以下指引内容：

```
📋 挑战日计划填写指引

1️⃣ 点击以下链接，填写计划单：
https://alidocs.dingtalk.com/i/nodes/Y1OQX0akWm6nkvZqubZnqjmqVGlDd3mE?utm_scene=team_space

💡 提示：建议在浏览器中打开（Chrome/Safari/UC），自动保存和截图功能更完整。如在钉钉内打开，点击右上角「···」→「在浏览器中打开」。

2️⃣ 填完点【一键复制发给助教】，把内容粘贴发给我

3️⃣ 我收到后会同步到钉钉AI表格【营运计划】，方便你后续复盘、汇报
```

**发送后等待店长填写并发回。**

### 步骤3：接收店长报告 → 解析 → 确认
店长发回报告有两种形式，都需要支持：

**形式A：文本报告（店长点击"一键复制发给助教"粘贴的文字）**
- 识别机器标记：`[JMF-CHALLENGE-DAY|门店|区域|日期|填表人|星期]`
- 从文本中提取：基础信息、目标汇总、品类详情、人员安排、促销活动、执行清单、当日复盘

**形式B：截图报告（店长点击"保存为截图"发送的图片）**
- 使用OCR识别截图中的文字内容
- 从OCR结果中提取上述所有字段

**解析后回复店长：**
> 收到！挑战日计划已解析：
> - 门店：XXX
> - 日期：XXXX-XX-XX（星期X）
> - 目标营业额：¥XX,XXX（环比+X%）
> 
> 确认无误后，我将同步到钉钉AI表格【营运计划】。

### 步骤4：同步到钉钉AI表格
店长确认后，将解析的数据写入钉钉AI表格。

**表格信息：**
- 表格链接：https://alidocs.dingtalk.com/i/nodes/Y1OQX0akWm6nkvZquvN2ABAjVGlDd3mE
- 子表名称：【挑战日计划】

**字段对应关系：**

| AI表格字段 | 对应挑战日计划数据 | 说明 |
|-----------|------------------|------|
| **门店** | 门店名称 | 基础信息 |
| **区域** | 所属区域（孙红梅区/陈桂莲区/陈超平区） | 基础信息 |
| **挑战日日期** | 日期（YYYY-MM-DD） | 基础信息 |
| **目标** | 目标营业额（上月/周同期 + 增量目标） | 自动计算 |
| **计划原文** | 完整的挑战日计划文本（一键复制的内容） | 包含全部详情 |
| **复盘** | 当日复盘（挑战日结束后填写） | 结束后更新 |

**同步后回复：**
> ✅ 已同步到钉钉AI表格【挑战日计划】！
> 
> 挑战日当天记得每2小时跟进进度，结束后把复盘结果发给我，我会更新到表格里。

### 步骤5：接收复盘更新（可选）
挑战日结束后，店长发送复盘数据（实际营业额、达成率、做对了什么、哪里没做到位、下次怎么做）。

AI将复盘内容更新到同一行记录的"当日复盘"字段中。

## 输出格式规则（钉钉适配）

- ❌ 禁止使用 markdown 表格（| 列1 | 列2 |）
- ❌ 禁止使用减号列表（- 项目）
- ✅ 使用 **加粗** 括关键信息
- ✅ 每行之间空一行（确保钉钉换行）
- ✅ 每段之间空一行
- ✅ 使用数字列表代替符号列表
