# 培训学前导读生成器

将培训文档转换为**学前快速导读页面**，让学员在正式学习前就能了解：
- 这堂课要讲什么
- 重点学什么
- 学完能收获什么

**核心定位**：学前5分钟速览，建立学习预期，不是完整课件。

---

## 使用流程

1. **接收文档**：用户上传 Word(.docx)、PDF(.pdf) 或 PPT(.pptx) 格式的培训文档
2. **提取内容**：使用 `scripts/extract_doc.py` 提取文档文本内容
3. **分析提炼**：
   - 识别培训主题和**学习目标**
   - 提炼**核心知识点**（3-5个，最多不超过7个）
   - 梳理**学习重点**和关键概念
   - 提取**学完后的收获/能做什么**
4. **生成HTML**：使用品牌模板生成简洁的导读页面
5. **输出文件**：直接输出单个 `.html` 文件

---

## 输出结构（精简版）

生成的HTML页面仅包含以下模块：

```
┌─────────────────────────────────┐
│  培训标题 + 一句话学习目标       │
├─────────────────────────────────┤
│  📋 学完你能做什么（3-5条）      │
│  （强调收获，而非内容罗列）       │
├─────────────────────────────────┤
│  🎯 核心内容速览                 │
│  - 要点1：一句话说明             │
│  - 要点2：一句话说明             │
│  - 要点3：一句话说明             │
├─────────────────────────────────┤
│  ⭐ 重点提示                     │
│  （学习时需特别关注的内容）        │
├─────────────────────────────────┤
│  💡 一句话总结                   │
└─────────────────────────────────┘
```

**总长度控制**：页面内容精简，5分钟内可浏览完毕。

## 工具使用

### 文档提取

```bash
python3 scripts/extract_doc.py <文档路径>
```

支持格式：.docx, .pdf, .pptx

### PPT内容以图片形式存储时的处理

如果PPT内容无法直接提取（文字以图片形式存储），使用OCR识别：

```bash
# 1. PPT转换为PDF
libreoffice --headless --convert-to pdf --outdir /tmp <pptx文件>

# 2. PDF转换为图片
pdftoppm -png <pdf文件> /tmp/pages

# 3. OCR识别（需要安装tesseract）
python3 -c "
import pytesseract
from PIL import Image
import os
for img in sorted(os.listdir('/tmp/pages')):
    if img.endswith('.png'):
        text = pytesseract.image_to_string(Image.open(f'/tmp/pages/{img}'), lang='chi_sim+eng')
        print(f'=== {img} ===')
        print(text)
"
```

**⚠️ OCR识别注意事项**：
- OCR识别可能存在错字（如"第二把火"识别成"第=把炙"）
- 需要人工核对关键数字、标题和数量
- 对于分多页讲述的同一主题，注意合并识别

### HTML生成

从模板 `assets/template.html` 开始，替换以下内容：
- `{{TITLE}}` - 培训标题
- `{{LEARNING_GOAL}}` - 一句话学习目标
- `{{OUTCOMES}}` - 学完能做什么（3-5条收获）
- `{{KEY_CONTENTS}}` - 核心内容速览（要点+一句话说明）
- `{{HIGHLIGHTS}}` - 重点提示
- `{{SUMMARY}}` - 一句话总结

### 输出

直接输出单个HTML文件，例如：`店长角色认知_导读.html`

## 内容提炼原则（精简导向）

1. **学前视角**：学员还没学，要告诉他"学完你会什么"，而不是"我要讲什么"
2. **极简原则**：每个要点一句话，不展开解释，不举例子
3. **收获导向**：多用"你将学会..."、"掌握..."、"能够..."
4. **控制长度**：
   - 核心要点：3-5条，最多7条
   - 每条：不超过20字
   - 总页面：5分钟内可浏览完毕

### ❌ 不要这样做

- 不要罗列所有章节
- 不要详细解释概念
- 不要长篇大论
- 不要包含具体案例

### ✅ 应该这样做

- 提炼最核心、最有价值的3-5个点
- 每点用一句话说明
- 强调"学完后的收获"
- 让学员产生学习期待

## 内容核对检查清单（必做）

在生成HTML之前，必须核对以下内容，避免遗漏：

### 数量核对
- [ ] 标题中提到的数量（如"5把火"、"6种角色"）是否全部提取？
- [ ] 分多页讲述的内容是否完整合并？
- [ ] 列表项是否有遗漏（如第2、3、4项）？

### 关键信息核对
- [ ] 核心理念/金句是否提取？
- [ ] 方法论/工具名称是否准确（如BEST反馈法）？
- [ ] 时间节点/数字是否准确（如24小时内、7天关键期）？

### 逻辑核对
- [ ] 要点之间是否连贯？
- [ ] 是否有明显的跳跃或缺失？
- [ ] 总结是否准确概括了全部内容？

### OCR特殊情况
- [ ] 识别出的乱码文字是否已修正？
- [ ] 关键术语是否核对原图确认？
- [ ] 序号是否连续（没有跳过）？

## 输出示例对比

### ❌ 原培训内容（详细版）

```
店长的6种角色：
1. 执行者 - 核心是把公司政策和总部任务有效落实，态度得是"先执行，再反馈"。公司上级安排的任务和要求，总能确保落地...
2. 品牌代言人 - 你的言行举止，就是品牌最直接的代言。在店里积极微笑，顾客更愿意停留...
...
```

### ✅ 学前导读版（精简版）

```
📋 学完你能：
✓ 清晰认识店长的6种核心角色
✓ 掌握门店经营的4大关键指标
✓ 从"执行者"思维升级为"经营者"思维

🎯 核心内容：
• 角色认知：执行者、代言人、经营者、问题解决者、领导者、教练
• 绩效指标：顾客满意、员工满意、营业额、利润
• 思维转变：从"自己做"到"带团队做"

⭐ 重点提示：
店长不是最能干的员工，而是最会带团队的经营者

💡 一句话总结：
6种角色、4大指标，核心是从执行者进化为经营者。
```
